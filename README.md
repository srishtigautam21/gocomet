## Features

- Product Listing Page
- Filters by:
   - Price
   - Gender
   - Brand
   - Sort By
     - Price : Low To High
     - Price : High To Low
     - Recommended items
     - New items
- Cart Management
- Wishlist Management

## Tech Stack

**Client:** React, Vanilla CSS
